Plateforme e-commerce avec système de réservation

HTML • CSS • JavaScript • Node.js/Express • PHP/PDO • EJS • API

Projet réalisé pour L'Art et La Manière ayant considérablement évolué au fil du développement. Initialement conçu comme un site vitrine avec système de réservation en ligne, il s'est enrichi d'une solution de paiement sécurisé, transformant le projet en une véritable plateforme e-commerce.

Note : Le projet comprend une partie confidentielle et n'est donc pas disponible publiquement sur github.

Fonctionnalités développées

  • Site vitrine avec rendu EJS sous Node.js/Express
  • Planning de réservation avec blocage des chevauchements côté serveur
  • Paiement en ligne via Stripe Checkout (forfait 8 séances et achat unitaire, création de session, enregistrement puis vérification du statut)
  • Interface d'administration
  • Galerie connectée à l'API Instagram (lazy-loading, gestion images/vidéos issus de carrousels, cache et fallback)

Architecture technique

  • Rendu EJS sous Node/Express et logique métier/API en PHP/PDO
  • Architecture claire et maintenable : séparation des responsabilités
  • Requêtes préparées, validations, codes HTTP explicites (par ex. 409 pour conflits de créneaux)
  • Documentation (README, scripts) et prêt pour la mise en ligne avec Docker

Démarche professionnelle

  • Approche agile avec livraisons régulières et versioning Git rigoureux
  • Prise en compte des référentiels OWASP/RGPD
  • Bonnes pratiques : validation des entrées, gestion sécurisée des sessions/cookies, accessibilité de base et performances maîtrisées

Note : Le projet n'a pas été mis en ligne, la cliente ayant changé d'avis.
